# GeoPy-Helfer

Ein einfacher Kommandozeilen-Geometrie-Helfer, geschrieben in Python, zur Berechnung von Flächen, Umfängen und Volumen gängiger geometrischer Formen.

## Funktionen

Dieser Helfer bietet Berechnungen für folgende Formen:

### 2D-Formen
*   **Kreis**: Fläche und Umfang
*   **Rechteck**: Fläche und Umfang
*   **Dreieck**: Fläche (basierend auf Grundseite und Höhe)

### 3D-Formen
*   **Würfel**: Volumen und Oberfläche
*   **Kugel**: Volumen und Oberfläche

## Installation

Es ist keine spezielle Installation erforderlich, da nur Standard-Python-Bibliotheken (`math`) verwendet werden. Stellen Sie sicher, dass Sie Python 3.x installiert haben.

## Verwendung

1.  Klonen Sie das Repository oder laden Sie die Dateien herunter.
2.  Navigieren Sie im Terminal in das Projektverzeichnis.
3.  Führen Sie die `main.py` Datei aus:
    ```bash
    python main.py
    ```
4.  Folgen Sie den Anweisungen im Menü, um die gewünschten Berechnungen durchzuführen.

## Projektstruktur

```
.
├── main.py
├── geometry_helper.py
└── README.md
```

*   `main.py`: Die Hauptdatei, die das interaktive Menü und die Benutzeroberfläche bereitstellt.
*   `geometry_helper.py`: Enthält alle mathematischen Funktionen zur Berechnung der geometrischen Eigenschaften.
*   `README.md`: Diese Dokumentation.
